# 国央企 Patterns

针对目标企业是中字头央企(国家电网、中石油、中烟、中铁、五大行总行、三大运营商、中投/中金等)及地方国企的客户。

---

## 风格关键词

**稳重 / 规范 / 完整 / 政治正确 / 长期主义**

国企招聘看重:**候选人能不能融入体系**、能不能**安心做事**、有没有**奉献精神**。跟大厂的"敏捷出活"是完全两种逻辑。

---

## 核心规则

### 规则 1:格式要工整、信息要完整

国企 HR 会**完整看完每一项**,所以:
- 个人信息要齐全(姓名 / 性别 / 政治面貌 / 籍贯 / 民族 / 出生年月都写)
- 教育背景**所有信息**都填(学校 / 专业 / 学位 / 学制 / 时间 / GPA / 排名)
- 实习经历按时间倒序,不要跳着选

大厂简历"删减留精华"在国企是反向操作:**信息越完整越好**。

### 规则 2:政治面貌一定要写

**党员是巨大加分项**。国企很多岗位会直接卡党员身份。
- ✅ 写在个人信息最显眼位置:"政治面貌:中共党员(YYYY 年 X 月入党)"
- 预备党员也要写
- 群众也要写,不写显得遮遮掩掩

### 规则 3:学生干部、党团活动一定要突出

国企看重"组织协调能力 + 集体观念"。这些经历放显眼位置:
- 学生会、团委、班委(具体职务 + 任期 + 工作内容)
- 党支部活动(主题党日、组织生活会、培训)
- 志愿者经历(总时长 + 服务对象)
- 三下乡、社会实践(尤其是党政机关、基层、贫困地区)

**这些在大厂简历里要删的,在国企简历里是核心加分项**。

### 规则 4:实习经历要突出"国家任务"和"重大项目"

国企背景的实习经历包装方向:
- 强调项目的"国家性"、"重大性"(eg 服务于 XX 战略、参与 XX 重大工程)
- 强调"配合上级"、"完成任务"的能力
- 不要太突出"个人英雄主义"和"创新颠覆"

模板:"参与 XXXX 项目(服务于 XX 战略 / 列入 XX 重点工程),负责 XX 板块,在 XX 老师指导下完成 XX 工作。"

### 规则 5:获奖荣誉要"主旋律"

国企 HR 偏好这些荣誉:
- 国家奖学金、国家励志奖学金 > 普通校级奖学金
- 三好学生、优秀团员、优秀党员、优秀学生干部
- 主旋律比赛(挑战杯、互联网+、创青春、大学生职业规划)
- 学科竞赛(数学/物理/化学/英语等国赛)

**不必突出**:商业比赛(eg 营销、商赛、案例大赛)在国企简历里权重不高。

### 规则 6:技能区写"通用 + 专业"

国企不太看花哨工具栈,看:
- **办公技能**:Office(熟练 Word、Excel、PPT)— 国企就要这种保守写法,不用合并
- **专业技能**:跟岗位强相关的(eg 财务用友/金蝶、法律 WK、工程 CAD)
- **语言**:英语六级、雅思,小语种特别加分
- **专业证书**:CPA、CFA、CCIE、教师资格证、法律职业资格证

**不要写**:Python、SQL、敏捷、Scrum、Figma 这种(显得不稳重,除非岗位明确要求)

### 规则 7:自我评价要"集体主义 + 长期主义"

国企爱看的自我评价关键词:
- 踏实、稳重、责任心强、抗压能力强
- 团队协作、组织纪律性
- 对 XX 行业有长期发展规划
- 愿意扎根、能吃苦

**避免**:创新、颠覆、追求自我实现、想做自己的事 — 这些在国企简历里是减分项。

### 规则 8:简历语气要"谦逊"

- ❌ "主导" → ✅ "参与 / 协助负责"
- ❌ "独立完成" → ✅ "在团队中承担 XX 工作"
- ❌ "颠覆 / 重塑" → ✅ "优化 / 改进"
- ❌ "极致量化" → ✅ "有具体数字但不夸张"

数字依然要有,但表述要稳。

## 国央企常见地雷

1. **政治不正确表述** — 涉及港台、历史、宗教问题的措辞要谨慎
2. **过度突出"商业成就"** — 国企不爱看"我赚了多少钱、我做了多大业务"
3. **学历断档 / GPA 不漂亮** — 国企政审严,简历漏洞会被查
4. **频繁跳槽** — 国企不爱要"不稳定"的人,跳槽多次要解释
5. **海外背景过强 + 没回国痕迹** — 国企有政审风险考虑

---

## 关于政审

国企岗位很多会做政审。准备国企简历时,你要注意:
- 家庭成员信息要真实(父母 / 配偶职业、政治面貌)
- 海外学习/工作经历要如实
- 出生地、籍贯要清楚
- 不要有任何虚假信息
