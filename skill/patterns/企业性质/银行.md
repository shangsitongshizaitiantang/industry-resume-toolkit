# 银行/金融机构 Patterns

针对目标企业是五大行(工农中建交)总分行、股份制银行(招商、平安、兴业、中信等)、外资行(汇丰、渣打、花旗、摩根、瑞银等)、证券、保险、基金、信托公司的客户。

---

## 风格关键词

**严谨 / 合规 / 风险意识 / 量化精准 / 专业背书 / 数字敏感**

银行金融招聘看重:**候选人专业不专业、严不严谨、抗不抗压、有没有合规意识**。比国企的"稳重"更专业,比大厂的"敏捷"更精确。

---

## 核心规则

### 规则 1:专业证书是硬通货

银行金融岗对证书极敏感。证书要放显眼位置(独立一栏在教育背景下面),按权重列:

**金融证书**:
- **CFA**(一级 / 二级 / 三级,通过几级写几级)
- **FRM**(风控类岗位)
- **CPA**(财务相关岗,投行 IBD 必备加分)
- **保代 / 证券从业资格 / 期货 / 基金从业**(投行/资管类)
- **CFP / AFP**(理财岗)
- **特许金融分析师助理 / CISI**

**附加证书**:
- **CET-6 580+ / 雅思 7+ / TOEFL 100+**(外资行必备)
- **CFA Level 1 通过(待考二级)**(还在考的也要写)

### 规则 2:量化必须精准、不能模糊

金融行业**数字就是命**。简历里所有数字必须:
- 单位明确(万 / 亿 / bps / pp)
- 时间维度明确(YoY / QoQ / MoM)
- 口径明确(净额 / 总额 / 加权平均)

示例:
- ❌ "提升了营收" → ✅ "推动 Q3 营收同比增长 12.3%(基期 X 亿元)"
- ❌ "降低了成本" → ✅ "通过 XX 模型将不良率从 1.8% 降至 1.2%(下降 60 bps)"
- ❌ "管理客户" → ✅ "管理 80+ 高净值客户,AUM 累计 5.2 亿元"

**数字不准是大忌**。你没确认的数据宁可不写。

### 规则 3:实习含金量分级

银行金融 HR 知道哪些实习是真有干货,哪些是水实习。要包装清楚:

**含金量高**:
- 投行 IBD 部门(并购、IPO、再融资项目)
- 自营 / 研究所(主动管理 / 卖方研报)
- 风控 / 信审部门(独立审单、模型搭建)
- 财务部门(参与年报编制、税务筹划)

**含金量中**:
- 客户经理 / 柜员(强调客户量、产品销售量)
- 二级市场投资助理(数据收集、报告整理)

**含金量低**:
- 大堂经理实习(打杂为主)
- 信用卡推广(地推)

**包装重点**:不管哪种实习,**必须挖出"专业动作"和"数据成果"**。即使是柜员实习,也能写"独立处理 200+ 笔现金业务,零差错率,服务投诉 0 起"。

### 规则 4:专业课程列出"金融相关"

教育背景下面,选择性列 3-5 门金融核心课:
- **必列**:公司金融、投资学、计量经济学、金融衍生品、风险管理、财务报表分析
- **加分**:固定收益、并购重组、私募股权、金融工程、量化投资

每门课**附 GPA / 成绩**(eg "公司金融 92/100, A")。这是银行 HR 验真才学的方式。

### 规则 5:行业项目要"模型化"

金融实习/项目描述模板:

> **项目背景**:XX 行业 / 公司估值需求(eg 拟 IPO 客户)
> **个人贡献**:负责 XX 模块,搭建 / 协助搭建 XX 模型(eg DCF / 可比公司估值 / LBO)
> **方法论**:使用 XX 方法,数据来源 XX(Wind / Bloomberg / Capital IQ)
> **成果**:输出 XX 报告 / 估值区间 XX 亿元,被客户 / 团队采纳

模型名词要专业:DCF / WACC / 多因子 / Black-Scholes / GARCH / VaR 等。

### 规则 6:外资行加分项

如果目标是外资行(汇丰、渣打、摩根、瑞银),额外要求:
- **英文简历必备**(中英双语都要)
- **海外经历加分**(交换、留学、海外实习)
- **第二外语加分**(尤其是粤语、日语、法语)
- **CFA / FRM / ACCA 比国内行更看重**
- **简历用 STAR 法写作**

### 规则 7:中资行更看"政治面貌 + 学校"

如果目标是中资行(尤其总行),五大行招聘逻辑接近国企:
- **党员加分**(尤其总行管培生)
- **985/211 学历是硬门槛**(部分行非清北复交不要)
- **学生干部经历突出**
- **政审过关**(参考国企 patterns)

## 银行金融常见地雷

1. **数字不精准** — 写"约 1 亿"显得不严谨,要么精准要么不写
2. **过度包装实习含金量** — 打杂的实习包装成"项目主导"会被面试一眼戳穿
3. **金融术语用错** — 把"市净率"和"市盈率"搞混这种,直接 reject
4. **没有量化** — 金融简历不量化等于没简历
5. **写"对金融充满热情"** — 空话,要换成具体动作(eg 自学 CFA / 关注 XX 行业)

## 自我评价 / 求职意向写法

**银行金融偏好**:
- 严谨细致 + 抗压能力 + 数字敏感 + 学习能力
- 对 XX 行业(eg 消费 / 医药 / TMT)有持续关注
- 通过 CFA X 级 / 自学 XX 课程

**避免**:
- 创新颠覆类词汇
- 不要谈"理想 / 情怀"
