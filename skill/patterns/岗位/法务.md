# 法务 / 律师 Patterns

涵盖:**企业法务(in-house) / 律师事务所律师(红圈所/外资所/精品所) / 合规 / 知识产权 / 诉讼 / 公司法务 / 数据合规**。法律行业有非常强的"圈层认知",简历的细节处理直接决定 HR 看不看下去。

---

## 风格关键词

**严谨 / 专业 / 精准 / 排版工整 / 客户思维**

法律行业看简历的人都受过法学训练,**对逻辑严密 + 措辞精准的容忍度很高、对马虎的容忍度极低**。

---

## 核心规则

### 规则 1:专业资格证书是绝对硬通货

法律岗简历**证书写在最顶端**(教育背景旁边或下面):

**中国法律**:
- **法律职业资格 A 证 / C 证**(LJQE,过了一定要写,通过几次写几次)
- **专利代理人**(知识产权方向)

**国际法律**:
- **JD / LLM / SJD**(国外法学学位)
- **NY Bar / Cal Bar / DC Bar / UK Solicitor / HK Solicitor**(外国律师资格)
- **CIPP / CIPM**(数据合规方向)
- **ACCA / CFA**(法务做财务合规方向加分)

附加加分:
- **CET-6 580+ / 雅思 7+ / TOEFL 100+**(外资所必备)
- **法律英语证书 / LEC**

### 规则 2:学校背景特别重要

法律行业认学历到极致。**国内 T0 法学院**(五院四系 + 北清复人浙交)是简历的核心 buff:
- 北大法学院 / 清华法学院 / 中国人民大学法学院 / 中国政法大学 / 西南政法 / 西北政法 / 华东政法 / 中南财经政法
- 复旦法学院 / 浙大光华法学院 / 上交凯原法学院 / 南大法学院

**海外**:Yale / Stanford / Harvard / Columbia / UChicago / NYU / Cambridge / Oxford / LSE 等。

学校好的话简历最顶部要突出。

### 规则 3:实习经历分级要清楚

法律行业的实习含金量分级**非常严格**:

**T0(顶级)**:
- 红圈所:金杜 / 君合 / 中伦 / 方达 / 国浩 / 海问 / 通商 / 竞天公诚 / 锦天城
- 外资顶级所:Davis Polk / Skadden / Sullivan & Cromwell / Cleary / Latham / Kirkland & Ellis 等
- 知名外资所中国办公室:Linklaters / Allen & Overy / Clifford Chance / Freshfields / Baker McKenzie

**T1(优秀)**:
- 大型综合所(汉坤 / 中伦文德 / 大成 / 国浩等)
- 知名精品所(汉坤资本市场 / 通商法律金融等)
- 大厂法务部(BAT / 字节 / 美团 / 腾讯法务部)
- 大型央企总法律顾问办公室

**T2(中等)**:
- 中型律所
- 中型企业法务部
- 大厂业务条线法务

**T3(基础)**:
- 小型律所
- 法院 / 检察院 / 仲裁机构实习

**包装重点**:实习等级低的,**必须挖深做过的具体项目**(eg "参与 X 亿美金跨境并购"、"独立撰写 Y 份合规审查报告")。

### 规则 4:项目要"案件级"具体

法律实习的描述必须具体到**案件 / 项目级**,不能用"协助处理日常事务"这种空话:

❌ "协助律师处理日常法律事务"
✅ "**参与 [客户名] 重大资产重组项目(交易金额 80 亿元)**,**负责尽职调查报告中财务/税务/劳动法部分(共 120 页)**,**协助完成对赌协议、股权代持协议等核心交易文件起草**。"

**写出**:
- 项目类型(并购 / IPO / 诉讼 / 仲裁 / 反垄断申报等)
- 项目规模(交易金额 / 标的额)
- 客户行业(尤其知名客户)
- 你的具体动作(尽调 / 起草 / 谈判 / 出庭等)
- 输出物(报告 / 协议 / 起诉状 / 备忘录)

### 规则 5:专业方向要明确

法律行业的细分方向决定职业路径,简历要表明**你想走什么方向**:

- **公司法 / 并购(M&A)**
- **资本市场(IPO / 再融资 / 债券)**
- **金融衍生品 / 监管合规**
- **争议解决(诉讼 / 仲裁)**
- **知识产权(专利 / 商标 / 著作权 / 反不正当竞争)**
- **反垄断 / 竞争法**
- **数据合规 / 隐私 / 网络安全**(2025 后特别热)
- **劳动法 / 雇佣**
- **税务**
- **破产清算**
- **房地产建筑工程**

简历的实习 + 课程 + 项目要围绕**目标方向**形成线索。

### 规则 6:课程 + 学术 + 模拟法庭

法律学生简历加分项:
- **核心课程**:列 3-5 门**跟目标方向相关**的课程 + 成绩
- **moot court / 模拟法庭比赛**:
  - Jessup(国际法模拟法庭)
  - Vis Moot(国际商事仲裁)
  - 中国"理律杯"
  - WTO 模拟法庭
  - 排名前列的写出来
- **学术发表**:文章/论文发表期刊,共同作者也可以写
- **研究助理**(给教授做 RA)

### 规则 7:语言能力是真实门槛

法律行业,尤其外资所 + 涉外法律:
- **英文必须流利**(简历英文版必备)
- **第二外语加分**(日语 / 法语 / 德语 / 韩语 — 涉外项目用得到)
- **法律英语**:能阅读 + 起草英文合同 + 用英文做法律分析

### 规则 8:title 重定位

法律岗的 title 体系比较固定:
- "法律实习生" / "法学院实习生" → 校招保持
- "律师助理 / 法务助理" → 保持
- "法务专员" / "法律顾问助理" → 保持
- "Paralegal" → 涉外简历用
- 律师从业后:"执业律师" / "高级律师" / "合伙人" → 别乱改

## 不同企业性质下的法务写法差异

| 企业性质 | 看重什么 | 写法重点 |
|---------|---------|---------|
| **红圈所 / 顶级外资所** | T0 学校 + 司考 + 英文 + 项目深度 | 大项目、跨境业务、专业资格 |
| **公司法务(大厂)** | 业务理解 + 商业 sense + 跨部门 | 数据合规、产品法律评审、风险管理 |
| **国企法务** | 体系完整 + 政治面貌 + 长期主义 | 党员、稳定性、央企业务知识 |
| **金融机构法务** | 合规 + 监管理解 + 严谨 | 资管、信托、监管报送、合规建设 |
| **精品所(知名律师工作室)** | 单一方向深度 + 学术能力 | 学术写作、出版物、专家网络 |

## 常见地雷

1. **简历有错别字 / 标点错误** — 法律人简历错字 = 即时翻篇
2. **排版凌乱** — 法律人极在意格式工整
3. **过度宽泛"参与处理"** — 必须落到具体案件 + 具体动作
4. **学校学历不突出** — 法律行业认学校,好学校要放显眼位置
5. **司考备考写在简历** — "正在备考法考"删,过了再写
6. **没有英文版** — 涉外法律岗位没有英文简历直接扣分

## 校招 / 实习法律特别建议

- **法考早过早写**(大三过法考是极强信号)
- **核心实习**集中在 T0 / T1 律所 / 大厂法务部
- **学术活动**:moot court / 论文 / RA / 法律援助
- **学生组织**:学生会法律部 / 模拟法庭协会 / 法律援助中心
- **英文标化**:法学院申请的同学,IELTS / TOEFL / LSAT 也要写

## 自我评价写法

**好的**:
> "中国政法大学法学硕士,**已通过法律职业资格 A 证**,3 段顶级律所实习(金杜 + 君合 + 中伦),专注**公司并购 + 资本市场**方向。在金杜实习期间参与 [客户] 跨境收购项目(80 亿元),独立完成 [120 页尽调报告]。**英文流利(雅思 7.5)**,可独立起草英文合同。"

**烂的**:
> "法学专业,严谨细致,有较强的法律思维和分析能力,擅长沟通和团队合作。"
