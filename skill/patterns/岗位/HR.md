# 人力资源 Patterns

涵盖:**HRBP / 招聘 / 培训发展(L&D)/ 薪酬绩效(C&B)/ 员工关系(ER)/ 组织发展(OD)/ HRIS**。HR 岗对"业务理解 + 数据 + 人际敏感度"的综合要求很高。

---

## 风格关键词

**业务理解 / 人际敏感 / 数据驱动 / 制度建设 / 跨部门协作**

不同子方向看重不同:
- HRBP:业务理解 + 跨部门
- 招聘:渠道 + 转化漏斗
- 培训:课程设计 + 评估体系
- 薪酬:数据 + 政策研究
- 员工关系:沟通 + 法律合规
- OD:组织诊断 + 变革管理

---

## 核心规则

### 规则 1:量化必须"HR 级精准"

HR 岗的核心 KPI 都能量化,要把数字写明白:

**招聘**:
- 招聘量(全年 / 月度 X 个 offer)
- 招聘周期(平均 X 天 / 同比缩短 Y%)
- 招聘漏斗(简历:面试:offer = X:Y:Z)
- 渠道效率(主流渠道转化率 / 内推占比)
- 雇主品牌(Net Hire / Offer Accept Rate)

**培训**:
- 课程数(年开发 X 门 / 覆盖 Y 人次)
- 满意度(NPS / 评分)
- 学习时长(人均 X 小时)
- 转化效果(课后行为改变 / 绩效改进)

**薪酬**:
- 涉及人数 / 总预算
- 调薪幅度(P50 / P75 分位)
- 内部公平性 / 外部竞争力指标

**HRBP**:
- 服务 BU 规模(覆盖 X 人 / Y 部门)
- 员工敬业度(年提升 X pp)
- 离职率(主动离职 / 整体)

示例:
> "**负责互联网行业 50+ 岗位招聘**,**全年完成 200+ offer 发放**,**平均招聘周期 28 天(行业平均 45 天)**,**通过内推占比从 18% 提升至 35%**。"

### 规则 2:业务理解要显式体现

HRBP / OD / 人才发展岗最看重"懂业务":

- "**深入电商业务**,熟悉双 11 / 618 大促节奏,**针对节假日峰值场景**优化兼职 + 全职配比模型"
- "**对接销售团队**,基于业务策略反推招聘 + 培训需求,**推动销售人效提升 X%**"

写出**你具体熟悉哪条业务线 + 用 HR 工具解决了什么业务问题**。

### 规则 3:制度 / 项目体系化经验

HR 简历最值钱的是"做过完整的项目 / 制度":

- "**搭建从 0 到 1 的校招体系**:涵盖 [校园品牌 + 渠道 + 测评 + 录用全流程]"
- "**主导薪酬带宽体系重构**:[市场调研 + 内部对标 + 落地方案 + 沟通培训] 4 个阶段"
- "**搭建管理者发展项目**:[需求调研 + 课程开发 + 师资 + 评估] 完整链路"

体系化经验 > 单点工作。

### 规则 4:HR 工具栈

HR 简历的工具栈分类:

```
招聘:LinkedIn / 智联 / 前程无忧 / Boss直聘 / 拉勾 / 内推系统 / Beisen / 北森
HRIS:Workday / SuccessFactors / SAP HCM / 北森 / 用友 NC / 飞书 People
数据 / 分析:Excel / Power BI / Tableau / SQL
测评:大五 / PDP / DISC / 霍兰德 / Hogan / 北森测评
培训:LMS 系统 / 课程开发工具 / Zoom / 腾讯会议
其他:Notion / 飞书 / Jira(项目管理)
```

**至少熟悉 2-3 个 HRIS / 招聘平台**,显得专业。

### 规则 5:法律合规意识

HR 岗,尤其员工关系 / 薪酬岗,**法律合规知识是隐形门槛**:

- "**熟悉《劳动合同法》《劳动法》《社会保险法》《个税法》**"
- "**处理过 5+ 起员工关系纠纷**,包括 [合同解除 / 仲裁 / 调解] 全流程"
- "**主导 [公司名] 劳动合同 / 员工手册修订**,符合最新法律法规"

### 规则 6:实习含金量分级

**含金量高**:
- 大厂(BAT / 字节 / 美团等)HR 部门
- 外企(P&G / IBM / Mars 等)HR / 招聘组
- 顶级猎头(Korn Ferry / Heidrick / Egon Zehnder / Spencer Stuart)
- 知名咨询公司 People & Org 业务线

**含金量中**:
- 中型企业 HR 部门
- 中型猎头公司
- 互联网创业公司 HR

**含金量低**:
- 校招中介 / 培训机构(纯销售性质)
- 小作坊 HR

**包装重点**:含金量低的实习要挖出"具体项目 + 量化结果",哪怕只是 1 个项目也比堆 5 个空泛职责强。

### 规则 7:title 重定位

- "HR 实习生" → 校招保持
- "招聘助理" → 校招保持
- "HRBP 助理" → 改成"HRBP(助理岗 / 培养项目)"
- "人力专员" → 看实际工作内容,改成对应子方向(招聘 / 薪酬 / 培训等)

## 不同企业性质下的 HR 写法差异

| 企业性质 | 看重什么 | 写法重点 |
|---------|---------|---------|
| **大厂(互联网)** | 数据 + 业务理解 + 高效率 | HRBP、人效 ROI、OKR 落地、敏捷招聘 |
| **国企 / 央企** | 体系完整 + 规范 + 政治正确 | 政审、党团活动、培训体系、稳定性 |
| **银行 / 金融** | 严谨 + 合规 + CPA / CFA 招聘经验 | 校招体系、风控合规、人才标准 |
| **外企** | 国际化 + 英文 + 系统性 | STAR、Talent Review、Succession Planning、SHRM |
| **咨询 / 猎头** | 客户思维 + Mapping + Sourcing | Account 服务、行业研究、Direct Search |

## 常见地雷

1. **简历像"职责清单"** — "负责 X / 协助 Y" → 没数据 / 没成果 → 翻篇
2. **过度强调"沟通能力强"** — HR 都说自己沟通强,要用具体案例
3. **不写业务理解** — 只写 HR 动作,不写懂什么业务 → HRBP 岗直接淘汰
4. **法律没case** — 写"熟悉劳动法"但没处理过案例 → 显得纸上谈兵
5. **测评 / HRIS 堆叠** — 列 10+ 工具不如 2-3 个深度使用

## 校招 / 实习 HR 特别建议

- **学生组织经历**特别加分(尤其学生会 / 校学生组织主席 / 团委)
- **校园招聘活动**做过的(校园大使 / 招聘宣讲会执行)很对口
- **HR 相关比赛**:HR 圆桌 / HR Pro 大学生赛 / 高校挑战赛 HR 分组
- **心理学 / 组织行为学背景**加分
- **实操过工具**:哪怕是社团用的简单招聘表格,也要写出来体现工具感

## 自我评价写法

**好的**:
> "人力资源管理专业,3 段大厂 HR 实习(腾讯 + 字节 + GM)。熟悉招聘 + HRBP + 薪酬三大模块,擅长用数据驱动 HR 决策。在字节实习期间独立完成 [项目],推动 [业务指标] 优化 X%。掌握 Beisen + Workday + Excel。"

**烂的**:
> "热爱 HR 工作,有沟通能力和亲和力,擅长团队合作。"
