# 销售 / BD Patterns

涵盖:**B 端销售 / C 端销售 / KA 客户经理 / BD(商务拓展)/ Account Manager / 客户成功(CSM)**。销售岗简历跟其他岗位非常不同,**核心是数字**。

---

## 风格关键词

**业绩硬数字 / 客户关系 / 行业理解 / 抗压能力 / 自驱力**

招聘官看销售简历就一件事:**你的业绩数字是多少**。其他都是辅助。

---

## 核心规则

### 规则 1:业绩数字必须出现在最显眼位置

销售简历最忌讳的是**不写数字**或**数字埋在后面**。每段经历的第一行就要出现业绩数据:

❌ "负责 XX 区域客户开发,维护重要客户关系。"
✅ "**全年业绩 1200w(任务完成率 138%,部门排名 TOP 3)**,负责华东区域 50+ KA 客户开发与维护。"

核心数字包括:
- **业绩总额 + 任务完成率 + 部门 / 公司排名**
- **同比增长 + 新客户开发数 + 客户保有数**
- **客单价 + 续费率 + ARR 增长**(SaaS 类)

### 规则 2:每段经历都要"业绩 + 方法"

光有数字也不够,要写出**怎么做到的**(让招聘官信你是真有方法论):

> "**全年新增 30+ KA 客户**,**贡献新签收入 800w(同比+45%)**。**通过 [行业研究 + 决策链梳理 + 多触达策略 + 高层关系建立] 四步法**,**将平均成单周期从 90 天缩短至 60 天**。"

业绩(What)+ 方法(How)= 完整销售故事。

### 规则 3:Pipeline 概念

成熟销售岗位看 pipeline 管理能力。简历可以写:

- "**管理 100+ 客户 pipeline**,**Lead → Qualified → Proposal → Won 转化率分别为 X% / Y% / Z%**"
- "**通过 CRM 系统(Salesforce)精细化管理客户阶段**,**Forecast 准确率达到 85%**"

### 规则 4:行业 / 客户类型要清楚

销售简历必须写明白服务过的**行业 + 客户规模**:

❌ "负责客户开发"
✅ "**负责互联网 + 金融行业 KA 客户开发**,**服务客户包括 [字节 / 美团 / 招行 / 平安] 等 5+ 头部品牌**"

✅ "**专注 SMB 中小企业市场**,**服务 200+ 客户,客单价 5-20w**"

让招聘官知道你**懂哪类客户、有没有 transferable 经验**。

### 规则 5:动词要"销售化"

销售简历的高质量动词:
- **开发**:开发 / 拓展 / 突破 / 挖掘 / 转化
- **维护**:维护 / 服务 / 经营 / 续费 / 增购
- **谈判**:洽谈 / 谈判 / 推动 / 达成 / 签约
- **管理**:管理 pipeline / 跟进 / Forecast / 复盘

避免:
- ❌ "参与" / "协助" / "了解"
- ❌ "负责" 后面没具体业绩

### 规则 6:工具栈要"销售人专属"

销售岗简历的工具栈:
- **CRM**:Salesforce / HubSpot / 销售易 / 纷享销客 / 飞书 CRM
- **数据 / Forecast**:Excel(必备 — 报表 / Forecast / 数据透视)
- **Sales Engagement**:Outreach / SalesLoft(外资销售必备)/ 火眼云
- **客户研究**:LinkedIn Sales Navigator / 天眼查 / 启信宝 / 同花顺
- **会议 / Presentation**:PPT / Keynote / Zoom / 腾讯会议

### 规则 7:Title 要清晰

销售岗 title 体系:
- "销售实习生" / "销售助理" → 校招保持
- "销售代表" / "Sales Rep" → 1-3 年
- "高级销售" / "Senior Sales" / "Account Manager" → 3-5 年
- "KA 客户经理" / "Senior AM" → 服务大客户
- "销售经理" / "Sales Manager" → 带团队
- "BD 经理" → 商务拓展(更偏战略合作)
- "客户成功" / "CSM" → SaaS 销售后的客户经营

校招 / 实习的销售岗,如果是 **GM(Graduate Management Trainee)/ 销售管培生**,要单独标出。

## 不同企业性质下的销售写法差异

| 企业性质 | 看重什么 | 写法重点 |
|---------|---------|---------|
| **大厂(互联网 / SaaS)** | ARR / 续费率 + 数据驱动 + 跨部门 | SaaS 指标、客户成功、CRM 精细化管理 |
| **快消 FMCG** | 渠道 + KA + 节点营销 | 大客户经营、新品上市、渠道下沉 |
| **制造 / 工业品** | 行业理解 + 长周期 + 大单 | 项目销售、招投标、客户关系深度 |
| **金融 / 保险** | 严谨 + 合规 + 高净值客户 | AUM、客户复购、专业证书 |
| **外企 B2B** | 英文 + STAR + Forecast + Pipeline | 国际化客户、Solution Selling、季度复盘 |

## 常见地雷

1. **写"业绩超额完成"但没数字** — 销售简历没业绩数字 = 没简历
2. **业绩数字不可信** — eg "完成率 500%" → 招聘官会怀疑,要补充任务量级和合理性
3. **客户名单堆叠** — 列 30 个客户名却没业绩归因 → 选 3-5 个最有代表性的写深
4. **过度强调"勤奋 / 抗压"** — 销售都说自己抗压,要用具体例子
5. **没有方法论** — 只写业绩不写怎么做的,招聘官会怀疑是吃运气

## 校招 / 实习销售特别建议

- **销售实习** > 营销实习(销售实习是真在做销售动作)
- **校园销售活动**:校园大使 / 课程销售 / 创业项目(校园咖啡店 / 二手交易)做过销售管理
- **比赛**:大学生创业大赛(商业落地组)/ 校园电商比赛
- **个人销售经历**:微商 / 代理 / 抖音直播带货(只要有真实数据)
- **应届进 KA / GM 项目**含金量最高

## 自我评价写法

**好的**:
> "市场营销专业,2 段 SaaS 公司销售实习(月业绩超额完成 30%,连续 3 个月 TOP 3)。熟悉 SaaS 销售全流程,精通 Salesforce 管理 pipeline,擅长用方法论复盘。Boss 直聘 + LinkedIn 主动找客户经验 100+ 人。"

**烂的**:
> "热爱销售,有抗压能力和沟通能力,适应能力强。"
