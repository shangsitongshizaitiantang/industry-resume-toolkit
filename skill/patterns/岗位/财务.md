# 财务 Patterns

涵盖:**会计 / 出纳 / 财务分析 / 财务 BP / 税务 / 资金 / 内审 / 财务规划(FP&A)**。审计岗见 `审计.md`(差异较大,单独列)。

---

## 风格关键词

**严谨准确 / 政策熟悉 / 数据敏感 / 业务理解 / 合规意识**

不同财务子方向看重不同:
- 会计 / 出纳:严谨 + 准确 + 流程
- 财务分析 / FP&A:业务理解 + 数据建模
- 财务 BP:跨部门 + 商业 sense
- 税务 / 内审:政策熟悉 + 风险意识

---

## 核心规则

### 规则 1:专业证书是硬通货

财务岗对证书极敏感,**证书要放显眼位置**(独立一栏在教育背景下面):

**中文证书**:
- **初级会计 / 中级会计 / 高级会计**
- **CPA(注册会计师)** — 通过几科写几科:"CPA(已通过 会计、审计、税法 3 科)"
- **CMA(管理会计师)**
- **税务师 / 审计师**

**国际证书**:
- **ACCA**(级别要标:F1-F9 完成 / SBL / SBR / 完整通过)
- **CFA**(级别)
- **USCPA**

**附加证书**:
- CET-6 580+ / 雅思 7+(外资财务岗必备)
- ERP 系统:SAP / Oracle / 用友 / 金蝶(熟练程度标清楚)

### 规则 2:量化必须"会计级精准"

财务岗简历的量化要**精准到单位 + 时间**:

- ❌ "提升了财务效率" → ✅ "通过流程优化,**月度结账周期从 7 天缩短至 3 天**"
- ❌ "管理客户应收账款" → ✅ "**管理 80+ 客户应收账款,日均处理 200+ 笔,坏账率 0.3%**"
- ❌ "做了财务报表" → ✅ "**独立编制月度财务报表(资产负债表 / 利润表 / 现金流量表),覆盖 5 家子公司**"

**金额单位明确**:万元 / 亿元 / RMB / USD,口径明确:净额 / 总额 / 已审 / 未审

### 规则 3:政策 / 准则要写出来

财务岗的专业度通过对政策的熟悉度体现:

- "**应用 CAS 14 收入准则**,推动公司销售合同收入确认重排,**避免审计调整 200w+**"
- "**熟悉 IFRS / GAAP / CAS 三套准则**,曾参与外资母公司合并报表编制"
- "**通过新个税专项附加扣除政策**优化高管薪酬结构,节税 100w+"

但**真懂的才写**,面试一问就漏。

### 规则 4:经历分类(财务子方向)

财务岗简历不要混着写,**按子方向分类**:

```
财务核算:负责日常账务处理 + 月结 / 年结
财务分析:负责经营数据分析 + 业绩追踪
财务 BP:支持业务线决策 + 跨部门沟通
税务:负责申报 / 筹划 / 税务合规
资金:负责现金流管理 + 融资支持
内审 / 风控:负责内部控制 + 风险评估
```

简历里**最多突出 2-3 个子方向**(不要写"什么都会"),展示你的核心专长。

### 规则 5:工具栈要分层

财务岗的工具栈分三层:

```
ERP 系统:用友 NC / 金蝶 K3 / SAP(熟练程度:精通 / 熟练 / 掌握)
办公工具:Excel(精通 — 数据透视 / VLOOKUP / 公式 / VBA / Power Query)
分析工具:Power BI / Tableau / SQL(财务分析 + FP&A 岗要求)
税务工具:增值税发票管理 / 防伪税控 / 金税系统
特殊工具:Bloomberg / Wind(资金 / 投资财务岗)
```

Excel 是基础,但**写"精通 Excel"要能说出具体到什么程度**(VLOOKUP / 透视表是基础,VBA / Power Query 才算精通)。

### 规则 6:业务理解(财务 BP 必备)

财务 BP / FP&A 岗简历最值钱的是"懂业务":

- "**深入互联网 SaaS 业务**,熟悉 ARR / MRR / NRR / 续费率等关键指标,**搭建 SaaS 商业模型**支持销售决策"
- "**对接销售 / 市场 / 产品 3 个团队**,提供月度财务复盘,**推动毛利率从 35% 提升至 41%**"

不只是会算账,而是用财务数据**帮业务做决策**。

### 规则 7:实习含金量分级

**含金量高**:
- 四大(PwC / EY / KPMG / Deloitte)审计 / 税务 / 咨询
- 内资八大(立信 / 信永中和 / 天职国际 / 致同 / 大华 / 中汇 / 中审众环 / 大信)
- 大型外企 / 上市公司财务部
- 投行 / 基金财务部

**含金量中**:
- 中型企业财务部
- 中型会计师事务所
- 财务咨询公司

**含金量低**:
- 个体户记账
- 小公司出纳(纯录数据)

**包装重点**:不管哪种实习,**必须挖出"具体项目 + 量化数据 + 学到的方法论"**。

### 规则 8:title 重定位

财务岗的 title 体系比较固定,不要乱改:
- "财务实习生" → 保持
- "会计助理" → 保持
- "出纳" → 保持
- 但小公司"财务专员"做的事如果有 FP&A 性质,可以写"财务专员(分析方向)"

## 不同企业性质下的财务写法差异

| 企业性质 | 看重什么 | 写法重点 |
|---------|---------|---------|
| **大厂 / 互联网** | 数据驱动 + 业务理解 + Excel/SQL | 财务 BP、SaaS 模型、AB 测试 ROI 计算 |
| **国企 / 央企** | 稳重 + 准则熟悉 + CPA | 准则应用、合规、长期主义、政治面貌 |
| **银行 / 金融** | CPA + 财务建模 + 风险量化 | DCF、估值、风险评估、行业研究 |
| **外企** | ACCA / IFRS / GAAP + 英文 | STAR 法、跨国合并报表、转移定价 |
| **会计师事务所** | 见 `审计.md`(单独列) | - |

## 常见地雷

1. **写"熟悉国内会计准则"但没具体案例** — 准则要落到具体业务上才有信号
2. **量化用"提升了 / 优化了"** — 没数字等于没说,要精准到百分比 / 金额
3. **ERP 系统堆叠** — 列 5 个 ERP 系统反而显得没专精,2-3 个熟练就够
4. **CPA 备考写法** — "备考 CPA"删,要写"CPA(已通过 X 科 / 备考中)"具体
5. **过度强调"细心"** — 财务人都说自己细心,要用具体例子(eg "零差错率 / 季度审计无调整事项")

## 校招 / 实习财务特别建议

- **CPA / ACCA 通过的科目**比 GPA 重要
- **暑期实习**含金量 > 学期内实习
- **企业财务岗实习**比"事务所审计实习"含金量高(尤其在企业财务方向)
- **校园经历**选跟数字相关的(eg 院级团委财经部、学生会财务部)

## 自我评价写法

**好的**:
> "会计学专业,CPA 已通过 4 科(会计、审计、税法、财管),2 段大型企业财务实习(SAIC + 字节)。熟练用友 NC + Excel,擅长财务分析与建模。在字节实习期间独立完成 [项目],推动 [业务指标] 优化 X%。中级会计 + ACCA F1-F9 完成。"

**烂的**:
> "财务专业出身,细心严谨,有责任心,擅长 Excel,能吃苦耐劳。"
