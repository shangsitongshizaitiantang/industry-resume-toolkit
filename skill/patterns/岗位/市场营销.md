# 市场营销 Patterns

涵盖:**品牌营销 / 数字营销 / 内容营销 / 公关 / 活动营销 / KOL 营销 / 联合营销 / Growth Marketing**。跨行业基础岗,但快消 / 新消费 / 互联网公司差异大。

---

## 风格关键词

**品牌感 + 创意 / 数据驱动 / Campaign 思维 / 跨部门协作 / 消费者洞察**

不同细分对侧重点要求很不一样:
- 品牌营销:看创意 + brand building
- 数字营销:看数据 + 投放效率
- 内容营销:看文案 + 流量
- 公关:看媒体资源 + 危机应对

---

## 核心规则

### 规则 1:Campaign 思维(分阶段讲)

市场营销最忌讳的是简历像"打杂流水账"。每个 campaign / 项目要分阶段讲:

> **策略阶段**:消费者洞察 / 目标设定 / 媒介选择 / 预算分配
> **执行阶段**:创意产出 / 渠道投放 / KOL 合作 / 活动落地
> **复盘阶段**:数据复盘 / ROI 测算 / 经验沉淀

每个阶段你具体做了什么,**让招聘官看到完整 campaign 思维**。

### 规则 2:量化必须挂"营销指标"

不同营销方向的量化指标:
- **品牌**:Awareness / Consideration / Preference / Top-of-Mind / NPS
- **数字营销**:CPM / CPC / CPA / CTR / ROAS / 转化率
- **内容营销**:阅读量 / 互动率 / 涨粉 / UGC 数 / 内容传播率
- **公关**:媒体覆盖数 / 报道传播量 / 负面舆情处理时效
- **活动**:参与人数 / 销售转化 / 媒体曝光 / Social Voice

示例:
> "策划'XX 节大促' campaign,**整合 15+ 媒介资源**,**KOL 矩阵覆盖 5000w+ 曝光**,**带动品牌专卖店 GMV 同比增长 38%**,**ROAS 达到 5.2**。"

### 规则 3:消费者洞察要显式写出来

营销岗的核心是"懂消费者",简历要显式体现:

❌ 不要写:"针对目标用户做了营销"
✅ 要写:"针对 [18-25 岁城市年轻女性] 在 [社交平台爆款焦虑] 痛点下的 [情绪释放] 需求"

**用户画像具体到 [年龄/性别/职业/收入/兴趣/场景]**,让招聘官看到你真懂消费者。

### 规则 4:跨平台 / 跨渠道经历是加分项

现在营销人最值钱的是"懂全渠道":
- **传统媒介**:报刊 / 电视 / 户外
- **数字媒介**:百度 / 字节系 / 腾讯系 / 阿里系信息流投放
- **社交平台**:小红书 / 抖音 / B 站 / 微博 / 微信
- **内容平台**:知乎 / 公众号 / 视频号
- **私域**:企微 / 社群 / 小程序
- **海外**:Meta / Google Ads / TikTok / KOL 海外

简历里写出**你跑过多少渠道**(2-3 个就够,不用堆全):
> "熟悉小红书 / 抖音 / 微信 三大平台的 营销 玩法,在 XX 项目中独立操盘小红书 KOL 投放,**100w 预算实现 800w+ 曝光、3000+ 私域转化**。"

### 规则 5:工具栈要"营销人专属"

营销岗简历的工具栈:
- **数据 / 投放**:百度推广 / 字节巨量引擎 / 腾讯广点通 / Google Ads / Meta Ads Manager
- **平台监测**:新榜 / 千瓜 / 灰豚 / 飞瓜 / 蝉妈妈 / Adobe Analytics
- **创意 / 设计**:Photoshop / Figma / 剪映 / Premiere
- **协作 / PM**:Notion / 飞书 / Trello / 项目管理工具
- **CRM**:Salesforce / HubSpot / 微盟 / 有赞

### 规则 6:Title 重定位(营销岗 title 很乱)

- "市场专员" / "市场助理" → 校招可以保留,社招改"市场营销"
- "BD" → 在 ToB 营销里可以归为"市场+销售"
- "策划" → 改成"营销策划 / 品牌策划"
- "公众号编辑" → 改成"内容营销 / 新媒体营销"
- "运营" 做 social media 的 → 改成"社交媒体营销"

### 规则 7:获奖 / 比赛是加分项

营销岗特别看重营销 / 商赛比赛经历:
- **L'Oreal Brandstorm**(欧莱雅校园营销大赛)
- **宝洁创新挑战赛**
- **联合利华未来领袖计划**
- **学院奖**(广告/创意类)
- **挑战杯 / 互联网+ **(商业组)
- **微博 / 抖音 / 小红书校园营销大赛**

获奖了一定要写,**没获奖但有参赛深度参与的也可以写**(eg "全国 50 强 / 区域决赛")。

## 不同企业性质下的市场营销写法差异

| 企业性质 | 看重什么 | 写法重点 |
|---------|---------|---------|
| **快消 FMCG(P&G/Unilever/L'Oreal)** | brand building + consumer insights | STAR 法、消费者深度洞察、品牌资产 |
| **新消费品牌** | 数据驱动 + 抢占心智 + 跨平台 | 私域、KOL 共创、节点营销、CPS |
| **互联网大厂** | 数据驱动 + 用户增长 | Growth Hacking、AB 测试、漏斗优化 |
| **奢侈品 / 高端品牌** | 品牌调性 + 圈层运营 | VIP 客户、活动质感、KOC 矩阵 |
| **B 端 SaaS / ToB 公司** | 内容营销 + lead generation | 白皮书、行业洞察、SEM、SDR |

---

## 常见地雷

1. **简历像"日记"** — 写"参与了 X 活动,做了 Y 事" → 没策略 / 没复盘 / 没结果
2. **量化只用 "曝光"** — 曝光数据最虚,要加上转化 / GMV / 互动率才有说服力
3. **跨平台堆叠** — 列 8 个平台显得啥都做但不专精,选 2-3 个写深
4. **创意主观描述** — "做了一个有创意的活动" → 要说服力靠数据,创意自己说没用
5. **没有 budget 概念** — 简历里写预算多少 + 怎么花的 + ROI,招聘官立刻知道你专业度

## 校招 / 实习营销特别建议

- **校园大使经历**特别加分(尤其是大厂的校园大使项目)
- **自媒体运营**:自己做过有真实数据的小红书 / 抖音 / 公众号
- **创业 / 副业**做过营销的:商品销售、IP 变现、咨询类
- **大学营销比赛**:深度参与的项目可以包装成完整 case study

## 自我评价写法

**好的**:
> "市场营销专业,3 段品牌营销实习(L'Oreal、字节、新消费品牌)。熟悉小红书 + 抖音双平台投放,在 [品牌方] 实习期间主导 [大促 campaign],100w 预算实现 800w+ 曝光、GMV 同比增长 30%。掌握巨量引擎、千瓜、Adobe AI。"

**烂的**:
> "对市场营销有浓厚兴趣,有创意和执行力,擅长团队协作。"
