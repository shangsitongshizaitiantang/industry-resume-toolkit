# 战略 / 咨询 Patterns

涵盖:**管理咨询(MBB / Big4 Strategy)/ 战略咨询 / 企业战略部 / 投资战略 / 行业研究 / Corp Dev / 商业分析**。这个赛道极度看重**结构化思维 + Top 学校 + Case Skills**。

---

## 风格关键词

**结构化思维 / 数据严谨 / 行业洞察 / Top 学校信号 / Leadership**

招聘官(尤其 MBB)看简历最在意:**这个人 case 面试能扛吗、有没有 leadership 潜力、英文够不够好**。

---

## 核心规则

### 规则 1:学校背景特别突出

战略 / 顶级咨询行业对学校极敏感:

**T0(MBB / Big 4 Strategy 直接看)**:
- 清华 / 北大 / 复旦 / 上交 / 浙大(管院/经院)
- 美国 Ivy + UChicago + MIT / Stanford
- 英国 LSE / Oxbridge / Imperial
- 欧洲 INSEAD / Bocconi / HEC / IESE

**T1(精品咨询 + Big 4 Advisory)**:
- 985 顶尖经管院校(中山 / 武大 / 厦大 / 南大 / 人大 / 央财 / 上财等)
- 海外 G5 / Top 30 美国高校
- 港大 / 港中文 / 港科大

学校好的话**学校放在简历最顶部**(可以加 GPA + 排名)。

### 规则 2:Leadership / Extracurricular 极重要

**咨询行业最关注"领导力潜力"**,所以简历的:
- **学生组织主席 / 团队负责人 / 项目负责人**
- **创业经历 / 自创项目 / 校园企业**
- **大型活动总策划**
- **community service / 志愿者领导岗**

这些**比成绩重要**,要放在显眼位置。

示例:
> "**清华大学经管学院学生会主席**(2023-2024,800+ 学生)**,主导年度迎新典礼策划**(覆盖 2000+ 新生),**带领 30 人执行团队**,**预算 10 万元**,**最终活动满意度 9.2/10**。"

### 规则 3:Case Skills 体现

简历要让招聘官看到你有 case 面试潜力:
- **结构化项目描述**(用 STAR / pyramid 结构)
- **量化所有可以量化的**
- **行业研究 + 商业分析项目**(写过的咨询报告 / 战略分析)
- **数据分析能力**(Excel 高级 + SQL + Python 基础)

### 规则 4:实习含金量分级严格

**T0(MBB 直接看)**:
- MBB(McKinsey / BCG / Bain)
- Big 4 Strategy(Strategy& / Deloitte S&O / EY-Parthenon / KPMG Strategy)
- 精品战略咨询(Roland Berger / OC&C / OW / LEK / Monitor)
- 顶级投行(GS / MS / JPM / UBS / Credit Suisse 等)的研究部 / IBD
- 顶级 PE(Carlyle / Blackstone / KKR 等)

**T1(咨询认可)**:
- Tier 2 咨询(埃森哲战略 / IBM 战略 / Capgemini Invent)
- 大厂战略部(BAT / 字节 / 美团 / 华为战略)
- 二线投行 / 中型 PE / VC
- 知名行研(中金 / 中信证券 / 国泰君安 研究所)

**T2(中等)**:
- 中型咨询公司
- 中型企业战略部
- 二级市场卖方研究

**包装重点**:写出**项目客户行业 + 项目类型(增长战略 / 运营改善 / 并购支持 / 数字化转型)+ 你的具体动作 + 输出物**。

### 规则 5:量化必须"咨询级"

战略 / 咨询岗的量化:
- **项目规模**:客户营收 / 资产规模 / 员工数 / 市场规模
- **战略影响**:为客户识别 / 节省 / 创造的价值($ amount)
- **运营改进**:效率提升 / 成本下降 / 周期缩短 / 体验改善
- **市场覆盖**:涉及行业 / 国家 / 客户类型

示例:
> "**为某 Fortune 500 消费品客户(中国营收 50 亿元)制定 2025-2030 增长战略**,**分析 5 个核心品类、12 个细分市场**,**识别 30 亿元增量空间**,**提出 8 项战略举措、3 个 quick win**。"

### 规则 6:英文必备

咨询行业(尤其外资)对英文要求极高:
- **英文简历必须备好**(STAR 法,1 页)
- **雅思 7+ / TOEFL 105+ / GMAT 700+**(申请研究生方向加分)
- **能用英文做 case interview**
- **第二外语加分**

### 规则 7:行业 / 职能方向

简历要表明你对**目标方向**有积累:

**行业**(写你深入的 1-2 个):
- TMT(科技 / 媒体 / 通讯)
- 消费品 / 零售
- 金融服务
- 医疗 / 制药
- 工业 / 制造
- 能源 / 电力
- 公共部门 / 社会服务

**职能**:
- 战略 / 商业模式
- 运营(精益 / Lean / 6 Sigma)
- 数字化转型
- 组织 / 人才
- 营销 / 客户体验
- 并购 / Corp Dev
- 风险 / 合规

简历的实习 / 项目要围绕**目标方向**形成主线。

### 规则 8:Case Competition / Hackathon

咨询岗看重:
- **case 比赛**:
  - **CFA Research Challenge**
  - **HSBC / DBS 商业 case competition**
  - **联合国 / 学院级 case competition**
  - **学校 Case Club**
- **行业商业比赛**:
  - **国际企业管理挑战赛(GMC)**
  - **欧莱雅 Brandstorm**
  - **宝洁创新挑战赛**

写出排名(Top X% / 决赛 / 一等奖等)。

## 不同企业性质下的战略 / 咨询写法差异

| 企业性质 | 看重什么 | 写法重点 |
|---------|---------|---------|
| **MBB / 顶级外资咨询** | T0 学校 + Leadership + Case Skills + 英文 | STAR、leadership story、case competitions |
| **Big 4 Strategy** | 学校 + 实习深度 + Industry sense | 行业研究、客户项目深度 |
| **大厂战略部** | 业务 sense + 数据 + 跨部门 + 落地 | 业务推动、跨团队、ROI |
| **企业内部战略部** | 行业理解 + 执行力 + 业务伙伴 | 行业深度、跨部门、长期 ROI |
| **金融机构战略 / 投资战略** | 估值能力 + 行业研究 + CFA | 金融建模、案例分析、专业证书 |

## 常见地雷

1. **简历像"任务清单"** — 战略 / 咨询岗最忌讳"做了 X, Y, Z" → 要有"为什么、怎么做、效果如何"完整故事
2. **量化缺失** — 战略简历没数字 = 没简历
3. **学校 + leadership 不突出** — 这两项是咨询行业绝对硬指标
4. **过度技术化** — 咨询岗简历不需要堆 30+ 工具,2-3 个核心就够(Excel + PPT + SQL)
5. **行业方向不清** — 写"对各行业都感兴趣"等于没方向
6. **case 面试经验未提** — 写明你练过 case(eg "完成 50+ 个 case 练习,接受过 [学长姐 / 培训机构] 辅导")
