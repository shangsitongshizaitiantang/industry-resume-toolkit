# 通用模式 - 跨行业、跨岗位

这里是不分行业岗位、所有简历都适用的改法模式。

---

## 模式

### 不放二维码,微信号用括号合并到电话后面

- **理由**:二维码占空间且打印版无法扫,微信号文字化更高效
- **示例**:`13500000000(微信同号)`,如果不同号就在姓名底下另起一行
- **适用条件**:所有简历

### 备考中的证书不写

- **理由**:备考≠拥有,写上反而暴露能力短板
- **示例**:"备考:CPA、雅思" → 删除。CPA 已经考过部分科目,可写成"CPA(已通过 XX、XX 科目)"

### Office 软件不拆开写

- **理由**:Word/Excel/PPT 是基本能力,拆开写显得在凑字数。专业工具(Vlookup、数据透视表)可以单独提
- **示例**:"熟练使用 Word、Excel、PPT" → "Office,熟练使用 Excel 的 Vlookup、Countifs 函数"

### 教育背景成绩要么全放要么不放

- **理由**:只放高分(91、93、87)等于在告诉招聘官"我有些课分数不好看"
- **示例**:原写"高级财务会计(91)、成本会计(93)、统计学(87)" → 要么补齐其他主修课分数,要么全删

### 课程列表写课程名,工具能力归到"计算机能力"

- **理由**:Excel、Python 等不是课程而是工具,放课程里显得搞混类别
- **示例**:课程列表里的"Excel"删除,放到"计算机能力"

### 获奖荣誉放在"教育背景"下面,不单立"校园经历-获奖荣誉"小节

- **理由**:荣誉是教育阶段成果,跟教育背景一起呈现更清晰、节省版面

### 量化必须具体,禁用模糊词

- **理由**:"很多""诸多""丰富""大量""多个"在简历里等于没说
- **示例**:"协助多家企业" → "协助 30+ 家企业";"处理大量数据" → "处理 2000+ 条数据"
- **量化维度**:时间 / 数量 / 团队规模 / 成果%/绝对值 / 覆盖范围

### 加粗小标题要短、要命中岗位关键词、删形容词

- **理由**:HR 阅读时先扫小标题,长标题=噪声,形容词=自夸
- **示例**:
  - "高效激活用户参与度" → "内容策划"
  - "精准定制宣发策略与高效执行" → "内容分发策略"
  - "主导专项资源库建设与体系搭建" → "体系搭建"
  - "成功拓展并落地艺人互动合作" → "艺人合作"

### 没有数据的地方坚决不瞎编,标 [需要你确认]

- **理由**:虚构数据是简历红线,被面试戳穿会失去全部信任
- **示例**:"独立完成 [需要你确认:多少笔] 细节测试性抽凭工作"
